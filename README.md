# DankMemer-Bot

Ce logiciel vous permet d'automatiser deux tâches du BOT DISCORD DANKMEMER ! 
Codé sur AutoHotKey !



PS : Il est préférable de ne pas changer de dossier ou de le renommer pour que le script marche niquel !
